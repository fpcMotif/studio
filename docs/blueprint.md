# **App Name**: OmniSearch Pro

## Core Features:

- Advanced Filtering: Provides multi-select dropdown menus for category, status, and tags, enhanced with search functionality.
- Diverse Input Options: Offers text input fields for main search query, optional date range, price range, and reference number inputs.
- Responsive Layout: Implements responsive design for optimal viewing on devices with breakpoints at 640px, 768px, and 1024px.
- Enhanced Usability: Includes form validation, debounced search, clear/reset functionality, and URL parameter persistence for form state.

## Style Guidelines:

- Primary color: A muted blue (#6699CC) to convey trust and stability.
- Background color: Very light blue (#F0F8FF), almost white, for a clean and modern look.
- Accent color: A contrasting orange (#FFA07A) for interactive elements to draw attention without being distracting.
- Use the default font
- Simple, outline-style icons from the chosen UI library, consistent with the form’s functionality.
- Consistent 8px grid system for spacing and alignment across all elements.
- Subtle animations for loading and error states to provide user feedback without being intrusive.
